# Tienda de Sweaters Acogedores - Módulo 2

Proyecto MVP frontend de e-commerce con Bootstrap y HTML semántico.

## Tecnologías
- HTML5 semántico
- Bootstrap 5 (grid, navbar, cards, responsive mobile-first)
- JavaScript básico (onclick inline para contador de carrito)

## Funcionalidades
- Grilla de 6 productos estáticos
- Contador de carrito en navbar
- Enlaces a detalle y carrito
- Responsive en móvil (≤420px) y desktop (≥1024px)

## Ver el proyecto
Abre `index.html` en el navegador.
